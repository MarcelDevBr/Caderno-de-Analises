# Caderno de Análises

**Conceito do Repositório**: Portfólio focado em ciência de dados e análise comportamental. Este repositório é estruturado de forma modular para hospedar diversos projetos independentes de modelagem, estatística e exploração de dados sobre a sociedade brasileira e outros temas de impacto.

Cada "Tema" funciona como um projeto analítico isolado com suas próprias bases de dados, scripts, notebooks e metodologias.

## 📂 Projetos Atuais

### [Tema 01: Comportamento Eleitoral 2026 - Bússola Eleitoral (VAA)](./Tema_01_Comportamento_Eleitoral_2026)
Desenvolvimento de um algoritmo de recomendação e análise de proximidade baseado na Teoria Espacial do Voto. Mapeamento de dores, demografia e capital cultural do eleitorado para a construção de um mapa político bidimensional.

### 🔜 Tema 02: (Pesquisa Futura)
*Próximos estudos e análises entrarão aqui como novos diretórios de temas.*

## 🏗️ Estrutura Geral do Repositório
- `Tema_01_Comportamento_Eleitoral_2026/`: Diretório completo contendo o primeiro projeto de análise (dados, notebooks, módulos e aplicação Streamlit).
- `Tema_XX_.../`: Novos projetos seguirão a mesma padronização de arquitetura.
- `requirements.txt`: Dependências gerais de Python utilizadas ao longo dos projetos.
- `.gitignore`: Arquivo para evitar subir dados sensíveis e gerados localmente para os projetos.
